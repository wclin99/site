# 2026-01-28

皇帝号装备的主炮为分布在五座双联装炮塔中的十门305毫米50倍径速射炮[注 3][4]。该舰摒弃了以往德式战列舰的低效六边形排列。相反，五座炮塔中的三座被安装在中心线上，其中设于舰艉的两座是以超射布局排列。其余两座则以雁列式布局安装在舰舯，可同向进行侧舷齐射[1]。副炮系统则包括十四门安装在舰舯炮廓内的150毫米45倍径速射炮、八门88毫米45倍径速射炮以及四门88毫米45倍径高射炮。此外，皇帝号还备有五具20英寸（508毫米）水下鱼雷发射管。其中舰艏一具，两边舷侧各两具[4]。
